FIGURELABS PROMPT  -  ARCHITECTURE OVERVIEW (ICERM POSTER, May 2026)
======================================================================

Use this prompt with FigureLabs to generate a publication-quality, fully schematic figure of the proposed agentic AI Science Community framework for automated neural operator discovery.

Style guide:
  - flat, geometric, infographic / scientific-publication style
  - clean white background, sans-serif (Helvetica / Inter / Source Sans)
  - color palette: TAMU maroon (#500000) for the framework backbone; accent blue (#1F4E79) for data flow; accent teal (#1ABC9C) for the citation graph; warm orange (#E67E22) for evaluator scoring; soft gray (#7F8C8D) for utility/connectors
  - block diagram look (not 3D, no gradients, no skeuomorphic shadows beyond a subtle drop shadow on cards)
  - single-image, landscape orientation, ~2.0 : 1 aspect ratio
  - text labels in small caps for headers, sentence case for body
  - target render at 300 dpi, vector-friendly so it can be embedded in a 48"x36" poster

Title (top-center, large, bold):
  An AI Scientific Community for Automated Neural Operator Discovery

Subtitle (smaller, italic, below title):
  16 virtual laboratories  -  citation-based economy of influence  - PSO swarm coordinator over architecture genomes

Layout (left to right):
  CENTER: a "swarm of laboratories" cloud
  LEFT INPUT panel: PDE family selector
  RIGHT OUTPUT panel: discovered neural-operator architecture

The figure must show four conceptual layers:

LAYER 1  -  PDE FAMILY (left edge, vertical strip)
  Show four small icons stacked vertically, one per PDE family the swarm
  has been applied to:
    1. "Piecewise Regression 1D"  - small icon: a step function with 4 jumps + a faint sinusoidal perturbation
    2. "Linear Advection 1D"      - icon: a triangle wave shifted right by an arrow
    3. "Burgers 1D"               - icon: a smooth sine that has steepened into a near-shock
    4. "Navier-Stokes 2D"         - icon: a vorticity field (2D blob of swirling colors, low-res)
  An arrow flows from this strip into the central swarm.

LAYER 2  -  THE SWARM (center, fills ~40% of figure width)
  Draw 16 circular "lab" nodes arranged in a loose cluster (NOT a grid).
  Each node is labeled "Lab i" (i=1..16), and each node carries a tiny pictogram of its current architecture: a horizontal mini-pipe of 4-6 tiny colored squares, where each color is a block type:
        red    = Fourier
        teal   = Attention
        green  = Wavelet
        orange = Residual Conv
        blue   = Branch-Trunk (DeepONet-style)
  Make sure no two labs have identical pictograms.

  Around each lab show a tiny five-spoked rosette indicating the five role agents inside it:
        Planner  | Worker  | Evaluator  | Reviewer  | Coordinator
  (small icons or initials P / W / E / R / C). The Coordinator agent is drawn larger and shared across labs in the center of the cloud (a hexagon labeled "Swarm Coordinator -- PSO + culling").

  Draw thin curved arrows BETWEEN labs in TEAL, representing citations (each lab cites its top-2 peers). Some arrows are thicker than others, suggesting that highly-cited labs accumulate trust. Show one or two "labs being culled" with a faded / ghosted appearance and a small "respawn" arrow from a high-trust lab.

  Below the swarm cluster, place one single equation in a callout box:
        tau_i^{t+1}  =  (1 - alpha) * tau_i^{t}  +  alpha * c_i / max_j c_j
  Caption: "trust score: decay + recent recognition"

LAYER 3  -  ARCHITECTURE GENOME (right of center, takes ~25% width)
  Show the OUTPUT of one selected lab (the global-best lab) as a horizontal pipeline of 7 colored blocks, in the order:
        Fourier -> Attention -> Wavelet -> Residual Conv -> Branch-Trunk -> Branch-Trunk -> Fourier
  Each block is a rounded rectangle in the matching block color, with the block name in white.
  Above this pipeline label: "Discovered hybrid (NS-2D, 1.0M params)"
  Below: small caption with the four families combined ("Spectral + Attention + Multi-scale + Operator-learning, no human prior")

LAYER 4  -  THE INNER GA (small inset, bottom-right of swarm cluster)
  A miniature flow showing what happens INSIDE a lab during one swarm iteration:
        Genome (parent)  ->  Mutation / crossover  ->  Train (50 epochs)
                          ->  Evaluate (rel L2 + efficiency + novelty)
                          ->  Composite fitness  ->  Cite peers
  Use small connected boxes with tiny arrows. Minimal text.

DATA FLOW ARROWS (top to bottom layer order):
  1. PDE family --> swarm  (single thick blue arrow, "input")
  2. Inside swarm: lab <--> lab citation arrows  (teal, bidirectional)
  3. Swarm coordinator --> all labs (maroon arrows, "PSO updates, spawns / culling")
  4. Global-best lab --> Architecture pipeline output (single thick orange arrow, "output: discovered neural operator")

LEGEND (bottom strip, horizontally):
  Five small color swatches with names:
        red = Fourier (FNO)        teal = Attention (Transformer)
        green = Wavelet (MWT)      orange = Residual Conv
        blue = Branch-Trunk (DeepONet)

Caption block (very bottom, single line, small italic):
  Reference: U. Braga-Neto, "The AI Scientific Community: Agentic Virtual Lab Swarms," arXiv:2603.21344 (2026).

Things to AVOID:
  - 3D perspective, gradients, glassy/glossy effects
  - photorealistic robots or anthropomorphic figures
  - cluttered text labels (keep labels short)
  - asymmetric chart-junk
  - any actual numerical curves; this is a CONCEPTUAL figure, not a results figure

Things to PRIORITIZE:
  - readability when shrunk to a poster column ~12 inches wide
  - flat, modern, scientific aesthetic (think: Anthropic / DeepMind / Nature Methods schematic figures)
  - the "swarm" should look organic but informative -- the viewer should immediately read the four layers (PDE -> swarm -> genome -> inner GA) without being told
